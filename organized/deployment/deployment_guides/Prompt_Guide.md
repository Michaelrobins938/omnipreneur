# 🧠 Prompt Guide: How to Deploy in ChatGPT, Claude, and OpenRouter

## ⚙️ GPT-4 (ChatGPT Pro)
**Best For:** Execution logic, structured SOPs, transformation arcs, funnel architecture  
**How to Deploy:**
1. Paste the full prompt into a new chat.
2. Begin with:  
   > "Phase 1: Scan Top 10 profitable eBook niches. Use BSR + Google Trends. Return with estimates and product links."  
3. Use follow-ups to initiate Phase 2–6.

## 🧠 Claude 3 Opus
**Best For:** Emotional tone, literary refinement, narrative voice, high-context summarization  
**How to Deploy:**
1. Start with:  
   > "Act as my narrative and emotional intelligence strategist. Refine the tone of this ADHD productivity workbook section…"  
2. Paste in writing samples, prompts, or outlines.

## 🛠️ OpenRouter (Multi-Agent Orchestration)
**Best For:** Multi-model chains with real-time validation  
**How to Deploy:**
1. Use the provided JSON agent config.
2. Assign:
   - GPT-4 = Tactical blueprint execution
   - Claude = Copy + narrative polish
   - Command R+ = BSR + competitive data
   - Mistral = Ad hook testing
3. Chain phases in Langchain, n8n, or a custom orchestrator.
