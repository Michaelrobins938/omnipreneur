## 🛠️ **Deployment Instructions**

### **1. Setting Up the Prompt Kit**

**For PDF Conversion:**
1. Open `Omnipreneur_Prompt_Kit.md` in any markdown editor
2. Export to PDF using your preferred method
3. Save as `Omnipreneur_Prompt_Kit.pdf`

**For Digital Use:**
1. Copy the markdown content directly
2. Use in any markdown-compatible platform
3. Reference specific prompts as needed

### **2. Setting Up the n8n Workflow**

**Prerequisites:**
- n8n instance (cloud or self-hosted)
- OpenRouter API key
- Notion integration (optional)
- Email service (optional)

**Import Steps:**
1. Open your n8n instance
2. Click "Import from File"
3. Select `Omnipreneur_n8n_Blueprint.json`
4. Configure environment variables:
   - `OPENROUTER_API_KEY`
   - `NOTION_PAGE_ID` (optional)
   - `EMAIL_FROM` (optional)
   - `DEFAULT_EMAIL` (optional)

**Configuration:**
1. **OpenRouter Setup:**
   - Sign up at openrouter.ai
   - Get your API key
   - Add to n8n environment variables

2. **Notion Integration (Optional):**
   - Create a Notion integration
   - Get your page ID
   - Add to environment variables

3. **Email Setup (Optional):**
   - Configure SMTP settings
   - Add email credentials to environment variables

### **3. Setting Up Lyra**

**For ChatGPT/Claude:**
1. Copy the entire content of `Lyra_FullStack_Prompt.txt`
2. Paste into a new chat
3. Start with "Hello" to activate Lyra

**For Custom Deployment:**
1. Use the prompt as a system message
2. Configure in your preferred AI platform
3. Set up as a custom assistant

## 🔧 **Troubleshooting**

### **Common Issues & Solutions**

**Problem:** n8n workflow not importing
**Solution:** Check JSON format and ensure all required fields are present

**Problem:** OpenRouter API errors
**Solution:** Verify API key and ensure sufficient credits

**Problem:** Lyra not responding as expected
**Solution:** Ensure the full prompt is copied and try restarting the conversation

**Problem:** Notion integration failing
**Solution:** Check page permissions and integration settings
