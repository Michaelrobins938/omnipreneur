Target platform: [Gumroad/Etsy/KDP]

## 📝 **PRODUCT DESCRIPTION GENERATION**

### **Claude 3 Opus Prompt**
```
You are a conversion copywriter specializing in digital products. Create a compelling product description for a digital bundle that converts browsers into buyers.

Product Title: [INSERT TITLE]
Files Included: [INSERT FILE LIST]
Target Platform: [Gumroad/Etsy/KDP]
Target Audience: [INSERT AUDIENCE]

Write a description that:
- Opens with a powerful hook that addresses the main pain point
- Lists 5-7 key benefits and features using bullet points
- Uses emotional triggers and power words
- Includes social proof elements (e.g., "Used by 10,000+ creators")
- Has a clear, compelling call-to-action
- Is 150-200 words total
- Is optimized for the specified platform's format
- Uses platform-specific terminology

Structure:
1. Hook (1 sentence)
2. Problem statement (1-2 sentences)
3. Solution introduction (1 sentence)
4. Key benefits (bullet points)
5. What's included (bullet points)
6. Social proof (1 sentence)
7. Call-to-action (1-2 sentences)

Make it sound urgent and valuable. Use emotional language that connects with the target audience.
```
