# 🚀 NOVUS: AI Prompt Optimization Agent

## What is NOVUS?

NOVUS is a master-level AI prompt optimization specialist that transforms vague, incomplete, or inefficient prompts into precision-crafted inputs that unlock AI's full potential across all platforms.

**Transform any rough idea into a tactical weapon with AI.**

## 🎯 What NOVUS Does

### The 4-D Methodology
1. **DECONSTRUCT** - Extract core intent, key entities, and context
2. **DIAGNOSE** - Audit for clarity gaps and ambiguity  
3. **DEVELOP** - Select optimal techniques and assign appropriate AI roles
4. **DELIVER** - Construct optimized prompts with implementation guidance

### Operating Modes
- **DETAIL Mode** - Comprehensive optimization with clarifying questions
- **BASIC Mode** - Quick fixes and lightweight prompt cleanup

1. Use `novus_protocol.md` for manual setup
2. Configure in your preferred AI platform
3. Follow the 4-D methodology for optimization

### Option 3: YAML Configuration
1. Import `novus_protocol.yaml` into compatible systems
2. Configure operating modes (DETAIL/BASIC)
3. Start optimizing prompts
