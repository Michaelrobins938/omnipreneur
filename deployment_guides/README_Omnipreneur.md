## 🚀 How to Use

### 🔁 Orchestrate the Stack
1. Use `novus_agent_blueprint.json` to assign Claude, GPT-4, Command R+ and Mistral roles
2. Run it via:
   - `n8n_omnipreneur_ai_chain.json` (No-code option)
   - `langchain_omnipreneur_chain.py` (Dev option)
3. All models route output to Notion Log or Live Dashboard

### 💼 Launch a Product
1. Use AutoRewrite or Content Spawner to ideate
2. Feed hooks into Bundle Builder
3. Use Aesthetic Generator to create Canva designs
4. Track rollout with Live Dashboard
5. Offer affiliate links via Affiliate Portal
6. Sell via Gumroad, Etsy, or KDP
