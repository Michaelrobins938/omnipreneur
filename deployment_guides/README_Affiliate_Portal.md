# 🌐 Affiliate Portal

Track affiliate referrals, generate links, and monitor stats (clicks, purchases, payouts).

## Pages
- `/pages/index.tsx`: Main portal UI
- `/api/affiliate/new.ts`: POST to generate referral link
- `/api/affiliate/stats.ts`: GET stats (mock)

## Components
- `Input`, `Button`, `Table` (Shadcn UI style)

## Setup
1. Paste files into Next.js project
2. Run: `npm run dev`
3. Open: `http://localhost:3000`
